# Copas

Shorten your long URL as eazy as Copy and Paste.  
URL Shortener service that easy to use, no need to signin, NO ADS and NO TRACKER built in.

![Thumbnail Image](/public/thumbnail.webp)

You can visit COPAS [here](https://copas-vanirvan.vercel.app/)

## references

The name **COPAS** is a shorthand from **COpy** and **PASte**  
This site is so easy to use as simple as copy and paste
